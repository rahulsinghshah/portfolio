# Rahul Singh Shah - Portfolio

A modern, high-performance portfolio website built for an AI & ML Engineer. The project showcases expertise in machine learning, full-stack web development, and data-driven solutions. 

## 🚀 Features

- **Sleek UI/UX:** Dark mode aesthetic with premium glassmorphism, dynamic animations, and clean typography.
- **Responsive Design:** Fully optimized for all screen sizes (mobile, tablet, desktop).
- **Project Showcase:** Highlights real-world ML and Full Stack projects with impact metrics and links.
- **Tech Stack Marquee:** A dynamic sliding pill-based marquee displaying core tools and technologies.
- **Interactive UI:** Features smooth hover effects, circular masked avatars, and modern Bento-styled grids.

## 🛠️ Technology Stack

- **Framework:** React with Vite
- **Styling:** Vanilla CSS tightly scoped with custom variables
- **Animations:** Lightweight CSS keyframes & transitions (optimized for performance)
- **Icons:** React Icons (`react-icons`)
- **Components:** `react-fast-marquee` for dynamic looping displays

## 📦 Running Locally

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The application will quickly start using Vite's optimized dev server.*

4. **Build for Production:**
   ```bash
   npm run build
   ```

## 🌐 Deployment Ready
The application is fully optimized. Run `npm run build` to generate the `/dist` folder, which can be instantly deployed to platforms like Vercel, Netlify, or GitHub Pages.
